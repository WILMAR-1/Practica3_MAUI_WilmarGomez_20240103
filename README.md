# Practica 3 - MAUI

Esta es una aplicación de demostración de .NET MAUI que explora diferentes layouts y características.

- **Formulario**: Un formulario de registro de usuario.
- **Dashboard**: Un panel con estadísticas y gráficos.
- **Galería**: Una galería de imágenes con avatares generados dinámicamente.
- **Perfil**: Una página de perfil de usuario.
