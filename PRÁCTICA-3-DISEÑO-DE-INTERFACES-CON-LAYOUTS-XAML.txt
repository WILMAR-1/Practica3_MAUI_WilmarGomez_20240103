OBJETIVOS DE LA PRÁCTICA

Al completar esta práctica demostrarás que:

   Dominas la sintaxis XAML para crear interfaces
   Puedes usar diferentes layouts según el diseño requerido
   Conoces las propiedades de espaciado y alineación
   Utilizas controles básicos (Label, Entry, Button, Image)
   Comprendes las diferencias entre XAML y XML de Android



DESCRIPCIÓN DE LA PRÁCTICA

Crearás una aplicación multi-pantalla que demuestre el uso de los 3 layouts
principales de MAUI: StackLayout, Grid y FlexLayout. Cada pantalla tendrá un
propósito específico y usará el layout más apropiado.

private void OnRegistrarClicked(object sender, EventArgs e)

{
    // Validar que todos los campos estén llenos

    if (string.IsNullOrWhiteSpace(entryNombre.Text))

    {

        labelResultado.Text = "   Por favor ingresa tu nombre";

        labelResultado.TextColor = Colors.Red;

        return;

    }



    if (string.IsNullOrWhiteSpace(entryEmail.Text))

    {

        labelResultado.Text = "   Por favor ingresa tu email";

        labelResultado.TextColor = Colors.Red;

        return;

    }



    if (!switchTerminos.IsToggled)

    {

        labelResultado.Text = "   Debes aceptar los términos";

        labelResultado.TextColor = Colors.Red;

        return;

    }



    // Si todo está bien

    labelResultado.Text = $"      ¡Bienvenido {entryNombre.Text}!";

    labelResultado.TextColor = Colors.Green;

}

P: ¿Puedo usar otros controles además de los mencionados? R: Sí, pero debes

CONSEJOS Y RECOMENDACIONES
   Para StackLayout:

   •   Usa Spacing para separar elementos uniformemente

   •   ScrollView es útil para formularios largos

   •   Orientation puede ser Vertical u Horizontal

   Para Grid:

   •   Define RowDefinitions y ColumnDefinitions claramente

   •   Usa Auto para filas/columnas que se ajustan al contenido

   •   Usa * para filas/columnas que ocupan espacio disponible

   •   Grid.ColumnSpan te permite que un elemento ocupe múltiples columnas

   Para FlexLayout:

   •   Wrap="Wrap" permite múltiples líneas

   •   JustifyContent controla la distribución horizontal

   •   AlignItems controla la alineación vertical

   •   Perfecto para galerías y diseños responsivos

   General:

   •   Usa Padding y Margin en múltiplos de 4 u 8

   •   Nombra elementos con x:Name si necesitas accederlos desde código

   •   Usa colores consistentes en toda la app

   •   Prueba en diferentes tamaños de pantalla



PROBLEMAS COMUNES Y SOLUCIONES

Problema 1: NavigationPage no funciona

Solución: Verifica en App.xaml.cs que tengas:

MainPage = new NavigationPage(new MainPage());

Problema 2: Grid no distribuye correctamente
Solución: Verifica que hayas definido RowDefinitions y ColumnDefinitions y que uses
Grid.Row y Grid.Column correctamente.

Problema 3: FlexLayout no hace wrap

Solución: Asegúrate de tener Wrap="Wrap" y que los items tengan WidthRequest
definido.

Problema 4: Validación del formulario no funciona

Solución: Verifica que los controles tengan x:Name y que el método esté en el
archivo .xaml.cs correcto.
